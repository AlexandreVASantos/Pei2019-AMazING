Switch Controller App






Use the grid buttons to enable or disable the switch PoE ports. The button represent the state of the port, which means if it's green and says ON the port is enabled, otherwise it is disabled.



If you pass the mouse right through node images you can see it's number and therefore know which node you are trying to turn on or off.



Nodes are constantly checking hardware conditions, if any alarm it's deployed you will se a notification like this.



If you click the bell in the top right corner you will be presented with a view like this. Here you will be able to see the new alarms sent by nodes when their hardware it is not in the best conditions or running properly. You can also filter by date to check all alarm with in an interval.
